Tentang Website

Informasi mengenai tujuan, latar belakang, dan teknologi yang digunakan dalam pembuatan Web GIS Purwokerto.

Tujuan Pembuatan

Website ini dibuat sebagai media untuk menampilkan dan memvisualisasikan hasil pemetaan wilayah Desa Purwokerto Wetan dan Kecamatan Purwokerto Timur. Peta disusun menggunakan perangkat lunak QGIS dan disajikan dalam bentuk Web GIS agar dapat diakses dengan mudah oleh pengguna.

Latar Belakang

Pemanfaatan Sistem Informasi Geografis (SIG) berbasis web sangat penting dalam penyajian data spasial yang informatif dan interaktif. Website ini dikembangkan sebagai bagian dari tugas akademik untuk menerapkan konsep SIG dalam bentuk aplikasi web.
